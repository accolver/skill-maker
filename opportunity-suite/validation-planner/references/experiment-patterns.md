# Experiment Patterns

Choose the lightest validation motion that can answer the main question.

## Common motions

- Warm intros + paid pilot
- Cold async outreach + problem interviews
- Audience pre-sell
- Marketplace listing test
- Waitlist + deposit
- Concierge delivery test
- Tiny self-serve product with paid checkout

## Matching guidance

- Use warm intros when trust matters
- Use audience pre-sell when distribution is already strong
- Use concierge tests when productization is unproven
- Use constraint-change plans when the right answer is effectively `not yet`
