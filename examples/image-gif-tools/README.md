# Image & GIF Tools

> **Status: Planned** — This skill is queued for construction with skill-maker.

A skill for working with images and GIFs: resizing, cropping, format conversion,
and creating GIFs from video or frame sequences. Combines `ffmpeg`,
`ImageMagick`, and `sharp` for fast, reliable media processing.

## When built, this skill will...

- Resize and crop images with precise control over dimensions and aspect ratios
- Convert between image formats (PNG, JPEG, WebP, AVIF, SVG, etc.)
- Create animated GIFs from video clips or sequences of frames
- Optimize images for web delivery (compression, stripping metadata)
- Batch process directories of images with consistent settings
- Generate thumbnails, sprites, and responsive image sets

## Tools & Dependencies

- `ffmpeg` — video-to-GIF conversion and frame extraction
- `ImageMagick` — general-purpose image manipulation
- `sharp` — high-performance Node.js image processing
