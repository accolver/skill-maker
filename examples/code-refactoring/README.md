# Code Refactoring

> **Status: Planned** — This skill is queued for construction with skill-maker.

A skill for code cleanup and refactoring: identifying code smells, extracting
functions, reducing complexity, removing duplication, and improving naming.
Provides systematic approaches to improving code quality without changing
behavior.

## When built, this skill will...

- Identify code smells (long methods, large classes, deep nesting, feature envy)
- Extract functions and modules to improve separation of concerns
- Reduce cyclomatic complexity through structured decomposition
- Detect and remove duplicated code across files and modules
- Improve variable, function, and class naming for clarity
- Apply refactoring patterns safely with test verification at each step

## Techniques

- Extract Method / Extract Class
- Replace Conditional with Polymorphism
- Introduce Parameter Object
- Decompose Conditional
- Rename for intent-revealing names
