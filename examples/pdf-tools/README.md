# PDF Tools

> **Status: Planned** — This skill is queued for construction with skill-maker.

A skill for working with PDFs: extracting text and tables, filling forms,
merging documents, and converting formats. Leverages tools like `pdftotext`,
`poppler`, and `pdf-lib` to handle common PDF workflows reliably.

## When built, this skill will...

- Extract text and structured table data from PDF documents
- Fill PDF form fields programmatically
- Merge, split, and reorder PDF pages
- Convert PDFs to and from other formats (images, HTML, etc.)
- Handle edge cases like scanned documents, encrypted files, and malformed PDFs
- Provide clear workflows for batch processing multiple files

## Tools & Dependencies

- `pdftotext` / `poppler` — text extraction and rendering
- `pdf-lib` — programmatic PDF manipulation
- `ghostscript` — format conversion and optimization
